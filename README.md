## ðŸ› ï¸ Setup & Customization
1. **Images:**  
   The current website uses placeholder images or Unsplash links. For the best result, take actual photos of your shop and products:
   - Save your hero image as `images/hero.jpg`.
   - Save product images in the `images` folder (e.g., `helmet.jpg`, `light.jpg`).
   - Update the `src` paths in `index.html` to point to your new local images.

2. **Google Maps:**  
   - Go to Google Maps, search for "Sri Mahadev Automobiles Sarjapura".
   - Click "Share" -> "Embed a map".
   - Copy the HTML (`<iframe>`) and replace the placeholder iframe in the `Contact Section` of `index.html`.

3. **Links:**
   - Update the WhatsApp link in `index.html` if the number changes.
   - Update social media links in the footer.
